# منصة توثيق قصص الصمود - غزة
مشروع مساق تطوير مواقع الويب 1 - جامعة الأقصى.

## فكرة المشروع
منصة إنسانية تهدف لتوثيق حكايات الأمل والصمود من قلب الميدان، باستخدام تقنيات الويب الحديثة لتعزيز الوعي الإنساني.

## التقنيات المستخدمة
* **HTML5:** لبناء الهيكل الإنشائي للموقع.
* **CSS3 & Bootstrap 5:** للتنسيق الجمالي وجعل الموقع متجاوباً مع الهواتف.
* **JavaScript & jQuery:** لعرض البيانات ديناميكياً (Arrays & Loops) وإضافة التفاعلية.

## ميزات المشروع
* قائمة تنقل علوية ثابتة (Sticky Navbar).
* عرض القصص باستخدام المصفوفات والدورات البرمجية.
* قسم خاص للتوثيق المرئي (الفيديو).
* تمرير سلس بين أقسام الموقع (Smooth Scroll).
